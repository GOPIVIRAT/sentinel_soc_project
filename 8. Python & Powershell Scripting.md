# PowerShell Essentials for a Cloud SOC Analyst

**ðŸ“Œ Why PowerShell Matters in Cloud Security**
- PowerShell is one of the most commonly abused tools by attackers
- Most modern attacks involve:
  - Executing malicious scripts
  - Downloading payloads using PowerShell
  - Performing recon (network, system, users)
  - Modifying registry or system settings
- Because attackers rely heavily on PowerShell, a SOC Analyst must understand PowerShell basics to:
  - Investigate alerts
  - Analyze processes
  - Identify suspicious activity
